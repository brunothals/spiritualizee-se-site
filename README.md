# Spiritualizee-se Site

Site público da Spiritualizee-se.

Esta pasta pode ser publicada no GitHub Pages, Netlify, Vercel ou Cloudflare Pages.

Arquivos:

- `index.html`: página pública
- `site.css`: identidade visual do site
- `site.js`: cena interativa da home

Não coloque o app de atendimentos neste repositório público.
